# ⚙️ Configuración del Proyecto

Esta guía explica cómo configurar el proyecto para desarrollo local y su puesta a punto inicial.

## 📋 Requisitos previos

Asegúrate de tener instaladas las siguientes herramientas:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (v18 o superior) y npm (se incluye con Node)
- Un editor de código (recomendado [VS Code](https://code.visualstudio.com/))
- (Opcional) [Docker](https://www.docker.com/) si el proyecto usa contenedores

   git clone <url-del-repositorio>
